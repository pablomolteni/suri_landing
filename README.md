# SURI Landing Page

Stack: React 18 + Vite + Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

## Build para producción

```bash
npm run build
```

## Deploy en Vercel

1. Subí este proyecto a un repositorio de GitHub
2. En vercel.com, importá el repo
3. Vercel detecta Vite automáticamente — solo hacé click en Deploy

## Estructura

```
src/
  App.jsx          # Componentes de toda la landing
  index.css        # Estilos globales + animaciones
  assets/logos/    # Logos de SURI
```

## Secciones

1. Navbar (sticky con scroll effect)
2. Hero (gradiente + CTA doble)
3. Problema (sección oscura)
4. Qué es SURI
5. Servicios (3 cards expandibles)
6. Cómo trabajamos
7. Para quién
8. Contacto + Footer
